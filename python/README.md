## ✨ Features

- Provides an enum of all well-known human languages (ISO 639-1).
- 100% pure and safe Python with zero dependencies and no bloat.
- Cuts red tape: 100% free and unencumbered public domain software.
- Polyglot software also available for Dart, Ruby, and Rust.

## 🛠️ Prerequisites

- [Python] 3.11+

## ⬇️ Installation

### Installation from PyPI

```bash
pip install -U known-languages
```
